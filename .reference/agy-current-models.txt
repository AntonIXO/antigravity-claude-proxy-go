Important routing facts:

- `gemini-3.5-flash-low` is the current upstream ID for the Medium tier. Its
  suffix is legacy/internal and is not the user-visible reasoning tier.
- `gemini-3-flash-agent` supplies the High tier that is not obvious from the
  raw ID.
- `gemini-pro-agent` is the selectable agent route for Gemini 3.1 Pro High.
  The separately published `gemini-3.1-pro-high` model is absent from
  `agentModelSorts` and Cloud Code rejects generation requests sent to it.
- The complete agent list comes from `agentModelSorts`, not every entry in the
  raw `models` object. The latter also contains image, tab, deprecated, and
  non-agent routes.
